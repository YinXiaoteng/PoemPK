# PoemPK
《诗词PK》App支持
-- 简介 --
《诗词 PK》是一款专为中小学生打造的古诗词学习应用，将经典诗词与趣味竞技深度融合，让孩子告别枯燥背诵，在欢乐的 PK 对决中爱上中华传统文化！
无论是唐诗宋词的千古绝唱，还是《诗经》的古朴风雅，都能在这里找到趣味学习的打开方式。
以赛促学的互动模式，既能点燃孩子的学习热情，更能帮助他们加深诗词记忆，轻松吃透课内课外的经典篇目。

-- 核心玩法・诗词对决大挑战 --
1. 分级闯关，精准适配
选择对应学习难度，系统随机抽取该级别经典诗词，以上句或下句为谜面，开启选字答题挑战。
2. 便捷操作，随心答题
内置重置、撤销、跳过功能，答题过程更灵活，轻松应对各种答题难题。
3. 双人同屏，实时对决
界面分为两个独立操作区，支持不同朝向的布局设计，满足 PK 双方同时作答的需求，面对面比拼更具沉浸感。
4. 动态比分，胜负立现
专属可移动圆形指示器，实时高亮当前答对题目数量领先的一方，战况一目了然。
5. 双重判定，快速决胜
规定时间内答对题目更多者获胜；若时间未结束，但一方答对数量已大幅领先对手，即可提前锁定胜局！
